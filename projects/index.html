People Counter

People Counter was a fun project to build. The basic idea is that a Time-of-Flight senser connected to a Raspberry Pi determins weather or not a person is crosing infront of it.

The Pi is acting as a server and a separate desktop is the web client. They communicate data to eachother using Socket connections.
